This folder contains the code for AES (courtesy of kokke) as well as the main.c testing file  
aes.h, aes.c, and main.c are used by Vitis  
constraints.xdc is used by Vivado and is based on the block diagram in the screenshots folder  
microblaze_system_wrapper.bit is created by Vivado and used as the hardware on the FPGA  
microblaze_system_wrapper.xsa is the hardware exported by Vivado for use by Vitis IDE to run software code.
