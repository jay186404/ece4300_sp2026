Folders:  
code: contains our source code  
demo_video: contains an MP4 screen recording of our program's output  
presentation: contains a PDF and PowerPoint file of our project presentation  
report: contains a PDF file of our documented report  
screenshots: contains screenshots of our program output as well as the block diagram from Vivado  

Software Required:  
AMD Vivado(for creating hardware on FPGA board)  
AMD Vitis(for writing software onto hardware created by Vivado)  

Vivado Blocks Used:  
- MicroBlaze V
- MicroBlaze Debug Module
- Clocking Module
- Processor System Reset
- AXI SmartConnect
- AXI Uartlite
- AXI Timer


Hardware Required:  
Digilent Nexys A7 100T FPGA Board
